O elemento de imagem (picture)

Redimensione o navegador para ver versões diferentes do carregamento da imagem em diferentes tamanhos de viewport. O navegador procura o primeiro elemento de origem em que a consulta de mídia corresponde à largura atual da janela de visualização do usuário, e busca a imagem especificada no atributo srcset.

O elemento img é necessário como a última tag filha do bloco de declaração de imagem. O elemento img é usado para fornecer compatibilidade com versões anteriores para navegadores que não suportam o elemento picture, ou se nenhuma das tags de origem corresponder.

Nota: O elemento picture não é suportado no IE12 e versões anteriores ou Safari 9.0 e versões anteriores.
