Imagens

A tag <img> é usada para inserir uma imagem em uma página HTML.
As imagens não são fisicamente inseridas em uma página da web;
as imagens são vinculadas a páginas da web.
A tag <img> cria um espaço onde a imagem será inserida.

Exemplo de código para imagem na web:


<img href="https://picsum.photos/200/300?random=1" alt="Imagens aleatórias" width="200" heigth="300">

Atualize a página para mudar a imagem.
