Google Maps

A incorporação do Google Maps é gratuita. Não é baseado em API. Basta copiar o código iframe e colá-lo no seu site. Apenas isso.
Para isso vá no site embedgooglemap.ne. Insira um endereço no campo de pesquisa à esquerda, selecione os detalhes de como deseja que seu mapa apareça como: foto de satélite ou mapa, zoom e tamanho da área onde será inserido. Clique no botão "Obter código HTML", irá aparecer um quadro com o código HTML. Copie esse código e cole no seu site onde deseja que o mapa apareça. Este é o código para tornar o código do mapa iframe totalmente responsivo: <style> iframe {width: 100%; height: 100%;} </style>.
Analise o código pois pode aparecer uma referência indesejado numa tag <a>, se aparecer remova o link.

Waze

Para incorporar o mapa ao vivo do Waze no seu site, use um iFrame, siga estas etapas:

  1. Abra sua página da Web em um editor de texto.
  2. Para gerar um iFrame na sua página da Web, adicione o código do iFrame a ela.
  3. Abra o Mapa em tempo real do Waze, busque o local na barra de pesquisa ou aumente o zoom e clique com o botão direito do mouse em um ponto específico do mapa.
  4. Um pop-up aparecerá com o nome da rua e algumas opções, clique em "compartilhar.
  5. Aparecerá um outro pop-up com o título "Compartilhar e embutir", clique na aba "incorporar um mapa" e em seguida o botão "COPIAR".
  6. Agora é só colar o código copiado no local do seu HTML onde ficará o iframe.
